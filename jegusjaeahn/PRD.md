# 회의 후처리 자동화 — Fireflies → 정리 → 공유 파이프라인

## 문제
> Fireflies로 회의 녹취·요약은 받지만, 정리해서 팀에 공유하는 과정이 수동이라 병목

- **현재 상태**: 회의 종료 후 Fireflies 요약을 수동으로 읽고, 액션아이템을 뽑고, Slack/Notion에 옮겨 적음. 건당 15~20분 소요
- **원하는 상태**: 회의 종료 후 스킬 한 번 실행으로 — 요약 + 액션아이템 추출 + Slack 공유까지 자동 완료
- **성공 기준**: 건당 후처리 시간 20분 → 3분 이내, 주 5회 이상 회의에 적용
